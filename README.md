# Shell-Script-Repo
A collection of shell script program .

## Run Locally

Clone the project

```bash
  git clone https://github.com/Souvik2376/Terminal-Clock.git
```

Go to the project directory

```bash
  cd folder-name
```

Give Permission

```bash
  chmod 777 programming_file.sh
```

Run

```bash
  ./programming_file.sh
```
